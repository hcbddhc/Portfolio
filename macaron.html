Macaron

Project Goal

To create an unique brand, and then to create a website using html and css.

Project Duration

  • 4 Weeks

Tool Used

  • Figma
  • HTML/CSS
  • Davinci Resolve
  • GIT

Project Type

  • Group Project

My Role

  • UI/UX Design
  • HTML/CSS Coding
  • Branding

The Macarons was a two person group project where the students were to create a unique brand from scratch and create a tailor made website for said brand.

We finalized with a local Macaron shop located in Montreal that specializes in making delicious macarons that come in many different flavours, the shop’s branding is accompanied by a cute mouse mascot. The website aims to best emphasize the company’s various macaron products which come in many different flavours. Additionally, the website aims to promote the company’s youthful, as well as classical nature. Therefore, creating a strong brand identity, and a website that can uphold that identity consistently is the key aspect to making this project successful.

I assisted in creating a suitable visual style that the website will be built around, before any feature was actually implemented for the site, through iterative and repeated experimentation and prototype building on figma. I’ve also taken initiative and planned out the page layouts and the site functions that would need to be included. More over, I’ve created an introduction video for the brand that would be used as a great first impression as the user enters the landing page. Additionally, I took on the role of being the primary coder in the later stage of development, responsible for replicating much of our final prototype design in html and css. As the project was developed in a git environment, I placed heavy emphasis on making the code clean and readable to prevent future problems and technical debt.

Everything combined, we created a phenomenal experience for users, from its aesthetics attracting users to find out more about the company and its macarons, to the seemingless and accessible process of buying the macarons, with the users being able to realize a strong and well defined company identity at the same time.
